## üßÆ Reward Formula

**Reward = BaseRate √ó MaterialMultiplier √ó Weight √ó EnvironmentalFactor**

- **BaseRate**: 10 tokens/kg  
- **Material Multipliers**:

- **Environmental Factor**: Dynamic (default = 1.0)
